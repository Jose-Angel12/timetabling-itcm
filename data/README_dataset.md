\# Dataset notes



This folder contains sample model instances used for reproducibility.



The original institutional database is not included because it may contain sensitive or private information.



Before public release, datasets should be anonymized by removing or replacing:



\- Professor RFCs or personal identifiers.

\- Student identifiers.

\- Institutional confidential fields.

\- Raw database credentials.

\- Full SQL dumps with private information.



The included JSON files are intended as reproducible model inputs for the MILP solver and dashboard examples.
